Free Trial characters can be carried over to the full game.
Switching to the full game not only lifts the level cap, but also enables a variety of useful features.

[Free Trial Preparation: PlayStation®5 | PlayStation®4]

  1. Game Installation
    First, install the FINAL FANTASY XIV client on your PlayStation®5 | PlayStation®4.

    1. Software Selection
      Search for FINAL FANTASY XIV on PlayStation Store™. Then select FINAL FANTASY XIV Free Trial and download it.

    2. Client Installation
      After downloading, the installation will start automatically.

  2. Game Procedures
    After the software installation is complete, start the software and follow the on-screen instructions. Now you're ready to play.

    1. Select FINAL FANTASY XIV displayed on the home screen.

    2. Carefully read the Software License Agreement screen and click "Agree" if you agree to the terms. FFXIV's launcher will now open.

    3. Select "Square Enix Account Holder" and enter the Square Enix account you created for the Free Trial.

    4. Please read the User Agreement and proceed upon acknowledging the terms.

    5. Confirm the details of the FINAL FANTASY XIV service account that will be created and the caution regarding the Free Trial, and then click "Next”.

    6. This concludes the contract for the Free Trial. Click "Play" to apply the latest patch. After applying the patch, press the "Play" button again to launch the game screen.

  3. Starting The Game
    You are now ready to play FINAL FANTASY XIV.

[Free Trial Preparation: Windows®]

  1. Game Download
    First, download the installer for FINAL FANTASY XIV to your computer.
    You can easily check if you can play the game comfortably on your computer with our official benchmark.
    Click here for the official benchmark

  2. Game Installation
    After downloading the client, install it on your computer.

    1. Setup
      Double-click the downloaded file "ffxivsetup_ft.exe" with your mouse to start the installer.

    2. Select Region/Language
      When the region/language selection screen appears, select the display language to be used during installation and in-game. Initially, it is the same as the region/language of your Windows® OS.
      ※ When installing the software, make sure you use an account with admin permissions.
      ※ If you select a language that is different from the language of your operating system, text may not be displayed correctly during installation.

    3. Note on other installed software/applications
      Please read the instructions on the screen carefully, as it contains information that you should be aware of when installing the software. If everything is OK, click “Next”.

    4. Selecting the installation location
      Select where you want to install the game data. If you want to change the installation destination, select "Browse" and specify the folder to install. When you have finished making your selections, click "Next”.
      ※ You must have at least 26GB of free space on the selected drive.
      ※ You must have at least 2GB of free space on the drive containing the My Documents folder.
      ※ You cannot use double-byte characters such as kanji, hiragana, and katakana in the name of the destination folder.

    5. Start the file copying process
      This will be the final confirmation of the installation. Check the installation location and if there are no problems, press "Next”. File copying will start.

    6. Complete the installation
      Click "Finish" to complete the installation. If you want to create a shortcut on the desktop, check the "Create shortcut on desktop" box.

  3. Game Procedures
    Once the client installation is complete, start the client and follow the on-screen instructions. You are now ready to play.

    1. Start FINAL FANTASY XIV.

    2. Carefully read the Software License Agreement screen and click "Agree" if you agree to the terms. FFXIV's launcher will now open.

    3. Select "Square Enix Account Holder" and enter the Square Enix account you created for the Free Trial.

    4. Please read the User Agreement and proceed upon acknowledging the terms.

    5. Confirm the details of the FINAL FANTASY XIV service account that will be created and the caution regarding the Free Trial, and then click "Next”.

    6. This concludes the contract for the Free Trial. Click "Play" to apply the latest patch. After applying the patch, press the "Play" button again to launch the game screen.

  4. Starting the Game
    You are now ready to play FINAL FANTASY XIV.

[Free Trial Preparation: Mac]

  1. Game Download
    First, download the installer for FINAL FANTASY XIV to your Mac system.

  2. Game Installation
    After downloading the client, install it into your Applications folder.

    1. Setup
      Double-click the download file "FINAL FANTASY XIV ONLINE.dmg" with your mouse.
      At this time, if Gatekeeper is enabled on your Mac, a confirmation message may appear.

    2. Installation
      When the window opens, drag and drop the icon for FINAL FANTASY XIV ONLINE that appears on the left side to the icon for the Applications folder that appears on the right side. The installation is now complete.
      Click "Play" to apply the latest patch. After applying the patch, press the "Play" button again to launch the game screen.

  3. Game Procedures
    Once the client installation is complete, start the client and follow the on-screen instructions. You are now ready to play.

    1. Double-click FINAL FANTASY XIV, which was installed from Launcherpad, to start it.

    2. Carefully read the Software License Agreement screen and click "Agree" if you agree to the terms. FFXIV's launcher will now open.

    3. Select "Square Enix Account Holder" and enter the Square Enix account you created for the Free Trial.

    4. Please read the User Agreement and proceed upon acknowledging the terms.
      Click "Play" to apply the latest patch. After applying the patch, press the "Play" button again to launch the game screen.

  4. Starting The Game
    You are now ready to play FINAL FANTASY XIV.
